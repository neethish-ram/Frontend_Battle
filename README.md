# AI Automation Platform

A premium responsive SaaS landing page built using React, Vite, and Tailwind CSS.

## Features

- Premium Hero Section
- Responsive Navigation
- Bento Grid (Desktop)
- Accordion (Mobile)
- State Persistence between Desktop & Mobile
- Matrix-Based Pricing Engine
- Monthly & Annual Billing
- Multi-Currency Support (USD, EUR, INR)
- Testimonials
- FAQ Section
- Responsive Footer
- SEO Friendly Semantic HTML
- Pure CSS Animations
- No External UI Component Libraries

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6)

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
